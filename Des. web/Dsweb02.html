Java: o que é?

Java é uma linguagem de programação de alto nível - linguagens que possuem um alto grau de abstração em relação aos códigos que um computador precisa interpretar para executar uma ação específica - uma das mais empregadas em dispositivos globais. Os seus programas estão disponíveis nos mais variados aparelhos, independentemente do sistema operacional que utilizam.

Trata-se de uma linguagem orientada a objetos, um dos paradigmas da programação, que se fundamenta na estruturação e parametrização de dados como atributos, que podem ser preenchidos e modificados, de forma resumida. Para mais informações, consulte este artigo. As aplicações desenvolvidas em Java também empregam a orientação por classes em seus procedimentos.

O propósito da criação da linguagem Java era reduzir a necessidade de implementar outras dependências durante o desenvolvimento de aplicações. Ainda podemos vê-la não somente como uma linguagem, mas como uma plataforma completa para o desenvolvimento dos mais diversos tipos de aplicações.

Atualmente, o Java é visto pelos programadores em atividade como uma ferramenta de extrema confiabilidade - considere que o Java está entre as linguagens e plataformas de desenvolvimento mais seguras disponíveis no mercado. A principal vantagem do Java reside na sua compatibilidade com uma variedade de sistemas e aparelhos.
