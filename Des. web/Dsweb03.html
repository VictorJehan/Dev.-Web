JavaScript: o que é?

JavaScript é uma linguagem de programação que possibilita a implementação de elementos complexos em páginas web. Sempre que uma página web vai além de apenas apresentar informações estáticas, apresentando conteúdo que se modifica em um determinado período de tempo, mapas interativos, gráficos 2D/3D animados, entre outros, é provável que o JavaScript esteja envolvido. Trata-se da terceira camada das tecnologias padrão web, das quais duas (HTML e CSS) discutimos em maior profundidade em outros segmentos da Área de Aprendizado.
